# Dataset reassembly

We do **not** redistribute raw datasets in this repo. Source licenses vary, and the PowerShell Corpus in particular ships without an explicit redistribution licence. This file tells you how to reassemble everything we use.

All processed artifacts (the deduped parquet, train/val/test splits, tokenizer vocab) are regenerated by `notebooks/02_data_assembly.ipynb` and hashed in `manifest.json`.

## Text-branch data (PowerShell scripts, ~15-20k labelled after dedup)

| Source | Location | What we take | Label |
|---|---|---|---|
| `das-lab/mpsd` | github.com/das-lab/mpsd | Benign + malicious + mixed-malicious (Fang 2021, Neurocomputing) | Labels preserved from source |
| `GhostPack/Invoke-Evasion` | github.com/GhostPack/Invoke-Evasion | `PowerShellCorpus.ast.csv.7z` (14,702 AST-feature samples), `BlackBoxData.ast.csv.7z` (3k samples) | Features only; labels from context |
| `Fa2y/Malicious-PowerShell-Dataset` | github.com/Fa2y/Malicious-PowerShell-Dataset | ~200 real-world malicious `.ps1` (SHA-256 filenames) | All malicious |
| `dessertlab/offensive-powershell` | huggingface.co/datasets/dessertlab/offensive-powershell | 1,127 NL-to-code pairs | Offensive examples |

**Deduplication:** SHA-256 of lowercased, whitespace-stripped script content.

**Output:** `data/processed/powershell_corpus_deduped.parquet` with columns:
- `sha256` тАФ dedup key
- `source` тАФ which dataset it came from
- `label` тАФ 0 (benign) / 1 (malicious)
- `script_text` тАФ raw script
- `first_seen_date` тАФ commit date for GitHub-sourced, first-seen for Malware Bazaar, `null` otherwise (used for Tesseract temporal splits)

We use **MAGIC's pre-processed `.pkl` graphs**, not raw CDM Avro. Avro parsing on Colab will OOM.

| Source | Location |
|---|---|
